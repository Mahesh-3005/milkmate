# 🥛 MilkMate - Dairy Management System

## 📌 Overview

MilkMate is a real-world mobile application built to manage daily operations of a dairy business handling **80+ customers**.
The app focuses on performance, reliability, and usability in **low-network environments** using an offline-first approach.

---

## 🚀 Features

* 👥 Customer management system
* 📦 Daily delivery tracking
* 💰 Income & expense management
* 🔄 Offline-first data handling (local + cloud sync)
* 📊 Scalable structure for future billing & analytics

---

## 🧠 System Architecture

* **Frontend:** Flutter
* **Backend:** Supabase (Auth + REST APIs)
* **Database:** PostgreSQL (cloud) + Hive (local storage)
* **State Management:** GetX
* **Architecture:** Offline-first with sync strategy
